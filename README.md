# fast-fastify

当前为 ts 为版本 js 版本为 jsBranch 分支 git clone -b jsBranch git@github.com:imbacc/fast-fastify.git

> 依赖于 fastify 框架，搭建架构!

## pnpm install

### 测试需要先创建数据库 -> 创建数据库名 test -> 导入 sql 文件夹下 \*.sql 语句执行

```
pnpm install && pnpm run dev
```
